1) git clone
- Que significa: sirve para copiar un repositorio de GitHub a mi ordenador
- Para que lo he usado: Para descargar mi repositorio p1-fork en mi ordenador y poder trabajar con él.

2) git status
- Que significa: Muestra el estado del repositorio y si hay cambios pendientes.
- Para que lo he usado: Para comprobar si había archivos nuevos o modificados antes de hacer commit
- Log de la terminal: 
On branch main
Your branch is up to date with 'origin/main'.

nothing to commit, working tree clean
